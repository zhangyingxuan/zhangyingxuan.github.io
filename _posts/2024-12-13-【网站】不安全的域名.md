1. 申请免费证书，有效期 3 个月；[腾讯云 ssl](https://console.cloud.tencent.com/ssl)
2. [手动解析/可选自动]添加 dns 解析，[域名解析](https://console.cloud.tencent.com/cns)2025-05-27 15:21:35 新增步骤
3. 审批通过后，下载证书，选择 nginx 方式，解压到本地

```shell
# 本地执行
scp -r /Users/sunny/Downloads/blowsysun.top_nginx root@43.154.209.141:/usr/local/etc
```

4. 登录服务器，进入目录`/usr/local/etc`，删除 blowsysun.top 目录（如果有）`rm -rf blowsysun.top`
5. 修改文件夹名称 `mv blowsysun.top_nginx blowsysun.top`
6. 修改 nginx 配置，cd 到`/etc/nginx/conf.d`;(如果文件名没有变化则不需要此步骤-blowsysun.top.pem)

```shell
# 服务器执行；
cd /etc/nginx/conf.d
vi blowsysun.top.conf
```

5. 重启 nginx `sudo nginx -s reload`
