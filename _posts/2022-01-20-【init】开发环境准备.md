nvm使用指令
```
nvm install node：安装最新版本的 node。
nvm install [<version>]：安装指定的 node 版本。
nvm ls：查看已安装的node 版本列表。
nvm uninstall <version>：卸载指定的 node 版本。
nvm use <version>：切换指定的 node 版本（临时）。
// 永久切换版本 ！！！
nvm alias default <version>：切换默认的 node 版本（永久）。
```
