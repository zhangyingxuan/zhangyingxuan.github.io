## 🔧 Chrome 插件作品

同时，作为一名 Chrome 扩展开发者，我致力于打造提升开发效率和用户体验的实用工具。

### 1. 🌐 网络请求拦截器

**Chrome 商店链接**: [网络请求拦截器](https://chromewebstore.google.com/detail/%E7%BD%91%E7%BB%9C%E8%AF%B7%E6%B1%82%E6%8B%A6%E6%88%AA%E5%99%A8/pcgmfkjckpfjhicmbajcjnkfkeconohc)

#### 核心功能

- **请求拦截与修改**: 实时拦截和修改 HTTP/HTTPS 请求
- **Mock 数据支持**: 创建和管理 API Mock 数据
- **性能监控**: 分析请求性能，识别瓶颈
- **调试工具**: 提供完整的网络调试功能

### 2. 📑 侧边栏 Tab 管理器

**Chrome 商店链接**: [侧边栏 Tab 管理器](https://chromewebstore.google.com/detail/%E4%BE%A7%E8%BE%B9%E6%A0%8Ftab%E7%AE%A1%E7%90%86%E5%99%A8/obgljjfbbjnigkpnmjbbdeoekemaieio)

#### 核心功能

- **智能分组**: 自动按域名对标签页进行分组
- **快速切换**: 侧边栏快速访问和切换标签页
- **搜索过滤**: 快速搜索和过滤标签页

### 3. 🍪 跨域同步 Cookies

**Chrome 商店链接**: [跨域同步 Cookies](https://chromewebstore.google.com/detail/%E8%B7%A8%E5%9F%9F%E5%90%8C%E6%AD%A5cookies/papfhjbfjcemehhoonkagdndlikahakl)

#### 核心功能

- **跨域同步**: 在不同域名间同步登录状态
- **安全传输**: 加密传输敏感数据
- **权限管理**: 精细控制同步权限
- **冲突解决**: 智能处理 Cookie 冲突
